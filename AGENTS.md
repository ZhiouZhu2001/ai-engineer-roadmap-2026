# Project Object

Project is about a CSV program, it will analyze a csv_path and read content, clean null values, calculate and categorize values.

# Structures

`main.py`: The program entry point, responsible for calling the reading, cleaning, analysis, and output functions.
`reader.py`: Reads CSV files.
`cleaner.py`: Handles missing values, outliers, and formatting issues.
`analyzer.py`: Calculates statistical results such as the number of occupations and average salary.
`tests/`: Contains test files.
`README.md`: Explains the project's purpose and how it runs.
`requirements.txt`: Records dependencies, such as pandas and pytest.

# Important

All code must be concise and clear to ensure performance and accuracy.
